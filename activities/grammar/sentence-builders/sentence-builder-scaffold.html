Adding Objects (What?)

Ask: WHO is doing it? WHAT are they doing? Add more information by including what the subject is doing something to.

Subject + V1/V1-3rd + Object
Positive
They pay bills.
Subject (They) + Verb (pay) + Object (bills) - No determiner needed (plural object)
Negative
They don't pay bills.
Subject (They) + Helper Verb (don't) + Verb (pay) + Object (bills)
Positive
She prepares a report.
Subject (She) + Verb (prepares) + Determiner (a) + Object (report) - Determiner needed (singular object)
Negative
She doesn't prepare a report.
Subject (She) + Helper Verb (doesn't) + Verb (prepare) + Determiner (a) + Object (report)
ðŸ’¡ When to Use Determiners:

Singular objects NEED a determiner: Use Determiner cards (a, an, the, my, your) before singular objects.

Plural objects DON'T need a determiner: Plural objects like "pizza", "books", "cars" can stand alone without a determiner.

Practice:
Build: Subject + V1/V1-3rd + Determiner (for singular objects only) + Object

Negative

Negative
He doesn't meet clients tonight.
Subject (He) + Helper Verb (doesn't) + Verb (meet) + Object (clients) + Time (tonight)

Time words can go at the beginning or end of the sentence. Time + Subject + Verb + Object also works!

Past Simple (V2)

Use past simple to talk about completed actions in the past.

Subject + V2 + Object
Positive
I paid the bills yesterday.
Subject (I) + V2 (paid) + Determiner (the) + Object (bills) + Time (yesterday)
Negative
I didn't pay the bills yesterday.
Subject (I) + Helper Verb (didn't) + V1 (pay) + Determiner (the) + Object (bills) + Time (yesterday)
Positive
She prepared a report last week.
Subject (She) + V2 (prepared) + Determiner (a) + Object (report) + Time (last week)
Negative
She didn't prepare a report last week.
Subject (She) + Helper Verb (didn't) + V1 (prepare) + Determiner (a) + Object (report) + Time (last week)
ðŸ’¡ Tip:

In negative past simple, use didn't + V1 (base form), not V2. The past tense is shown by "did", not the main verb.

Past simple often uses time words like yesterday, last week, last night, or specific past dates.

Practice:
Build: Subject + V2 + Time

Subject + will + V1 + Object
Positive
I will pay the bills tomorrow.
Subject (I) + will + V1 (pay) + Determiner (the) + Object (bills) + Time (tomorrow)
Negative
I won't pay the bills tomorrow.
Subject (I) + won't + V1 (pay) + Determiner (the) + Object (bills) + Time (tomorrow)
Positive
She will prepare a report next week.
Subject (She) + will + V1 (prepare) + Determiner (a) + Object (report) + Time (next week)
Negative
She won't prepare a report next week.
Subject (She) + won't + V1 (prepare) + Determiner (a) + Object (report) + Time (next week)
ðŸ’¡ Tip:

Future simple uses the base form of the verb (V1) after "will". Use "won't" or "won't" for negatives.

Future simple often uses time words like tomorrow, next week, next weekend, or in the future.

Use continuous tense to talk about actions happening right now or actions in progress.

Subject + To Be + V1-ing + Object

Subject (I) + To Be (am) + Verb-ing (preparing) + Determiner (a) + Object (report)
Negative
I'm not preparing a report.
Subject (I) + To Be (am not) + Verb-ing (preparing) + Determiner (a) + Object (report)
Positive
She is closing the shop.
Subject (She) + To Be (is) + Verb-ing (closing) + Determiner (the) + Object (shop)
Negative
She isn't closing the shop.
Subject (She) + To Be (isn't) + Verb-ing (closing) + Determiner (the) + Object (shop)
ðŸ’¡ Tip:

Match the To Be verb to your subject: I am, You/We/They are, He/She/It is

Subject + To Be + V1-ing + Object + Preposition + Place + Time
Positive

Subject (I) + To Be (am) + Verb-ing (preparing) + Determiner (a) + Object (report) + Preposition (at) + Determiner (the) + Place (office) + Time (today)
Negative
I'm not preparing a report at the office today.
Subject (I) + To Be (am not) + Verb-ing (preparing) + Determiner (a) + Object (report) + Preposition (at) + Determiner (the) + Place (office) + Time (today)
Positive
She is closing up the shop tonight.
Subject (She) + To Be (is) + Verb-ing (closing) + Preposition (up) + Determiner (the) + Object (shop) + Time (tonight)
Negative
She isn't closing up the shop tonight.
Subject (She) + To Be (isn't) + Verb-ing (closing) + Preposition (up) + Determiner (the) + Object (shop) + Time (tonight)
ðŸ’¡ Tip:

Present continuous often uses time words like now, at the moment, or right now to show it's happening currently.

8

Past Continuous

Use past continuous to talk about actions that were in progress at a specific time in the past.

Subject + To Be (was/were) + V1-ing + Time or Connector + second event
ðŸ’¡ Important Rule:

Affirmative past continuous sentences usually need a second event or time reference. The past continuous action is often interrupted by another action, or it happens at a specific time.

Examples with Connecting Words (Two Events):

Positive
I was cooking when he came home.
Subject (I) + To Be (was) + Verb-ing (cooking) + when + Subject (he) + Verb (came) + Object (home)
Negative
I wasn't cooking when he came home.
Subject (I) + To Be (wasn't) + Verb-ing (cooking) + when + Subject (he) + Verb (came) + Object (home)
Positive
She was preparing a report before the meeting started.
Subject (She) + To Be (was) + Verb-ing (preparing) + Determiner (a) + Object (report) + before + Determiner (the) + Object (meeting) + Verb (started)
Negative
She wasn't preparing a report before the meeting started.
Subject (She) + To Be (wasn't) + Verb-ing (preparing) + Determiner (a) + Object (report) + before + Determiner (the) + Object (meeting) + Verb (started)
Positive
They were meeting clients while I was checking emails.
Subject (They) + To Be (were) + Verb-ing (meeting) + Object (clients) + while + Subject (I) + To Be (was) + Verb-ing (checking) + Object (emails)
Negative
They weren't meeting clients while I was checking emails.
Subject (They) + To Be (weren't) + Verb-ing (meeting) + Object (clients) + while + Subject (I) + To Be (was) + Verb-ing (checking) + Object (emails)
Positive
He was driving to the office as the rain began.
Subject (He) + To Be (was) + Verb-ing (driving) + Preposition (to) + Determiner (the) + Place (office) + as + Determiner (the) + Object (rain) + Verb (began)
Negative
He wasn't driving to the office as the rain began.
Subject (He) + To Be (wasn't) + Verb-ing (driving) + Preposition (to) + Determiner (the) + Place (office) + as + Determiner (the) + Object (rain) + Verb (began)

Examples with Time Reference:

Positive
They were meeting clients at 3 PM yesterday.
Subject (They) + To Be (were) + Verb-ing (meeting) + Object (clients) + Time (at 3 PM) + Time (yesterday)
Negative
They weren't meeting clients at 3 PM yesterday.
Subject (They) + To Be (weren't) + Verb-ing (meeting) + Object (clients) + Time (at 3 PM) + Time (yesterday)
ðŸ’¡ Connecting Words for Past Continuous:

You can use these words to connect past continuous with another event:

  • when - shows interruption (I was cooking when he came home)
  • while - shows two actions happening at the same time (I was working while she was calling)
  • as - shows two actions happening at the same time (I was leaving as she arrived)
  • before - shows the continuous action happened before another event (I was preparing before the meeting started)
  • after - shows the continuous action happened after another event started (I was working after they left)
  • until - shows the continuous action continued up to a point (I was waiting until he arrived)
  • during - shows the continuous action happened at the same time as another event (I was working during the meeting)
ðŸ’¡ Tip:

Match the To Be verb to your subject: I/He/She/It was, You/We/They were

Remember: Past continuous often describes an action that was happening when something else occurred, or shows what was happening at a specific past time.

Practice:
Build: Subject + To Be (was/were) + V1-ing + Time or connecting word (when, while, before, after, as, until, during) + second event

Future Continuous

Use future continuous to talk about actions that will be in progress at a specific time in the future.

Subject + will + be + V1-ing
Positive
I will be preparing a report at 3 PM tomorrow.
Subject (I) + will + be + Verb-ing (preparing) + Determiner (a) + Object (report) + Time (at 3 PM) + Time (tomorrow)
Negative
I won't be preparing a report at 3 PM tomorrow.
Subject (I) + won't + be + Verb-ing (preparing) + Determiner (a) + Object (report) + Time (at 3 PM) + Time (tomorrow)
Positive
She will be meeting clients next week.
Subject (She) + will + be + Verb-ing (meeting) + Object (clients) + Time (next week)
Negative
She won't be meeting clients next week.
Subject (She) + won't + be + Verb-ing (meeting) + Object (clients) + Time (next week)
ðŸ’¡ Present Continuous vs Future Continuous for Future Events:

Present Continuous for Future: Use when the future event is planned, arranged, or scheduled (often with a specific time).

I am meeting clients tomorrow at 2 PM.
This is a planned/scheduled meeting

Future Continuous: Use to describe what will be happening at a specific time in the future, or to show an action in progress in the future.

I will be meeting clients at 2 PM tomorrow.
This emphasizes the action will be in progress at that specific time

Key difference: Present continuous focuses on the plan, future continuous focuses on the ongoing action at a future time.

ðŸ’¡ Tip:

Future continuous often uses future time words like tomorrow, next week, at 3 PM, or in the future.

Practice:
Build: Subject + will + be + V1-ing + Time

Negative Statements

Simple Present Negative:

Subject + Helper Verb (don't/doesn't) + V1 + Object
I don't pay bills.
Subject (I) + Helper Verb (don't) + Verb (pay) + Object (bills)

Present Continuous Negative:

Subject + To Be (am not/isn't/aren't) + V1-ing + Object
I am not preparing a report.
Subject (I) + To Be (am not) + Verb-ing (preparing) + Determiner (a) + Object (report)
She isn't closing the shop.
Subject (She) + To Be (isn't) + Verb-ing (closing) + Determiner (the) + Object (shop)

Past Continuous Negative:

Subject + To Be (wasn't/weren't) + V1-ing + Object
I wasn't cooking when he came home.
Subject (I) + To Be (wasn't) + Verb-ing (cooking) + when + Subject (he) + Verb (came) + Object (home)

Future Continuous Negative:

Subject + won't + be + V1-ing + Object
I won't be preparing a report tomorrow.
Subject (I) + won't + be + Verb-ing (preparing) + Determiner (a) + Object (report) + Time (tomorrow)

Past Simple Negative:

Subject + Helper Verb (didn't) + V1 + Object
I didn't pay the bills yesterday.
Subject (I) + Helper Verb (didn't) + V1 (pay) + Determiner (the) + Object (bills) + Time (yesterday)

Future Simple Negative:

Subject + won't + V1 + Object
I won't pay the bills tomorrow.
Subject (I) + won't + V1 (pay) + Determiner (the) + Object (bills) + Time (tomorrow)
ðŸ’¡ Tip:

In negative sentences, use the base form of the verb (V1), not V1-3rd or V2, after don't, doesn't, didn't, or won't.

Summary: Key Formulas

Simple Present:

Subject + V1/V1-3rd + Object + Time

Present Continuous:

Subject + To Be (am/is/are) + V1-ing + Object + Time

Past Continuous:

Subject + To Be (was/were) + V1-ing + Time or Connector + second event

Past Simple:

Subject + V2 + Time

Future Simple:

Subject + will + V1 + Time

Future Continuous:

Subject + will + be + V1-ing + Time
ðŸ’¡ Remember:
  • Simple Present = habits, routines, facts
  • Present Continuous = actions happening now OR planned future events (scheduled/arranged)
  • Past Simple = completed actions in the past
  • Past Continuous = actions in progress in the past (usually needs a second event or time reference)
  • Future Simple = actions that will happen in the future
  • Future Continuous = actions that will be in progress at a specific future time (emphasizes ongoing action)
  • Use V1-3rd (adds -s/-es/-ies) with He/She/It in simple present
  • Match To Be verb to your subject: I am/was, You/We/They are/were, He/She/It is/was
  • For future events: Present Continuous = the plan/schedule, Future Continuous = the ongoing action at that time
