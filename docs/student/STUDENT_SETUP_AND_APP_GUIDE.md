# Student Setup and App Guide

Use this guide when you join class for the first time.

## 1. Sign In for the First Time

1. Open the class website link from your teacher.
2. Tap **Login**.
3. Enter your:
   - Username
   - Temporary password
4. Tap **Sign In**.

## 2. Set Your New Password

On first login, you will be asked to create a new password.

1. Enter a new password you can remember.
2. Confirm the same password.
3. Tap **Save**.

Tips:
- Do not share your password.
- Use at least 8 characters.
- Write it in a safe place if needed.

## 3. Join Your Class Section

1. Go to **Dashboard**.
2. Tap **Join Class**.
3. Enter your class code from your teacher.
4. Tap **Join**.

If you see **Class not found**, check:
- You typed the code exactly.
- No extra spaces.
- You are using the correct section code.

## 4. Save to Home Screen (Use Like an App)

## iPhone / iPad (Safari)

1. Open the site in **Safari**.
2. Tap the **Share** button.
3. Tap **Add to Home Screen**.
4. Tap **Add**.

## Android (Chrome)

1. Open the site in **Chrome**.
2. Tap the **3-dot menu**.
3. Tap **Add to Home screen** or **Install app**.
4. Tap **Install / Add**.

## 5. Daily Use

1. Open the app from your home screen.
2. Check **Weekly Checklist**.
3. Complete activities and quizzes.
4. Check your class leaderboard (your section only).

## Quick Troubleshooting

- **Can’t log in**: Ask teacher to reset your password.
- **Wrong class/section**: Ask teacher for the correct class code.
- **App not updating**: Close app and reopen after a minute.
- **No internet**: Reconnect and refresh.
