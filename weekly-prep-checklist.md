# Weekly Prep Checklist
## ESOL Level 3 - Spring 2026

Use this checklist each week to ensure you're ready for Tuesday and Thursday classes.

---

## Week of: _____________ (Week #: ____)

### Before Tuesday (Grammar Lesson Day)

**Materials**
- [ ] Print Tuesday warm-up worksheet (2-sided if possible)
- [ ] Print any handouts for grammar lesson
- [ ] Queue up grammar guide on Class Companion (test that it loads)
- [ ] Prepare game materials (cards, dice, timer, etc.)

**Content Review**
- [ ] Review this week's grammar topic - what are the key points?
- [ ] Review warm-up worksheet - any tricky vocabulary to pre-teach?
- [ ] Check teaching schedule for R/L/W task - what text/audio needed?
- [ ] Plan 2-3 example sentences for grammar modeling

**Tech & Admin**
- [ ] Check which students completed last week's grammar guide
- [ ] Review verb quiz results (due Tuesday 6pm) - common errors?
- [ ] Update attendance from last class
- [ ] Post any announcements to class

---

### Before Thursday (Review + Phonics Day)

**Materials**
- [ ] Print Thursday warm-up worksheet
- [ ] Prepare phonics/pronunciation materials
- [ ] Prepare vocabulary retrieval activity (matching cards, etc.)
- [ ] Print unit quiz if scheduled this week

**Content Review**
- [ ] Review Tuesday's grammar - what did students struggle with?
- [ ] Plan error correction examples from Tuesday's work
- [ ] Check phonics focus for the week (see teaching schedule)
- [ ] Plan 2-3 minute vocabulary retrieval routine

**Student Work**
- [ ] Collect Tuesday's warmup writing (if assigned)
- [ ] Grade/review any collected work
- [ ] Note students who need extra support

---

### End of Week

**Wrap-Up**
- [ ] Update any student progress notes
- [ ] File/organize this week's materials
- [ ] Note what worked well / what to change
- [ ] Check next week's schedule - any special prep needed?

**Looking Ahead**
- [ ] Next week's grammar topic: _______________________
- [ ] Any materials to create/find: _______________________
- [ ] Unit quiz coming up? Date: _______________________

---

## Quick Reference

### Tuesday Class Flow
1. Speaking warm-up (10-15 min)
2. Quick spiral review + set goal (5 min)
3. Grammar lesson - NEW (25-30 min)
4. R/L/W task (20-25 min)
5. Speaking/grammar game (15-20 min)
6. Exit ticket + homework reminders (5 min)

### Thursday Class Flow
1. Speaking warm-up (10-15 min)
2. Grammar review + short practice (15-20 min)
3. Phonics/pronunciation (10-15 min)
4. R/L/W task (20-25 min)
5. Speaking/grammar game (15-20 min)
6. Wrap-up (5 min)

### Vocabulary Retrieval Ideas (2-3 min Thursday)
- Quick matching (word → definition)
- "Define it without saying it"
- Charades with vocab words
- Category sort
- Sentence completion race
