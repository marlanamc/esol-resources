Part C: Partner Practice

5 min
Round 1: Share what you used to do at a previous job. What was different?
Round 2: Switch. Discuss: "What are you still getting used to at work?" "What can't you get used to?"

Remember: "used to" = past habit, "be used to" = accustomed, "get used to" = becoming accustomed.

Reading

Work changes can be difficult. When policies change, you need time to adjust. "Used to" describes past habits: "I used to work days." "Be used to" means you're comfortable now: "I'm used to the schedule." "Get used to" means you're still adjusting: "I'm getting used to the new system."

These three structures help you talk about changes at work.

1) What is the difference between "I used to" and "I'm used to"?
2) What does "I'm getting used to" mean?
3) What is something at work that you are still getting used to?

Writing Homework

6–8 sentences
Prompt: Describe changes at your current or past job. What did you use to do? What are you used to now? What are you still getting used to?
  • Try using "used to" (past habit)
  • Try using "be used to" (accustomed)
  • Try using "get used to" (adjusting)
  • You might include vocabulary from Part A

Optional challenge: Use "would" for repeated past actions ("I would walk to work every day").
